
# IT1214-Student-Grade-System

## Project Purpose

The Student Grade Management System is a Java console-based application developed to help teachers manage student records digitally.

The system allows users to:

* Add student records
* Display all student records
* Search for a student using Student ID
* Calculate the average marks of all students

This project demonstrates Object-Oriented Programming (OOP) concepts and the use of GitHub for version control.

## Features

1. Add Student
2. Display All Students
3. Search Student by ID
4. Calculate Average Marks
5. Exit Program

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Git & GitHub

## How to Run the Program

### Compile the Program
javac StudentGradeManagementSystem.java

### Run the Program
java StudentGradeManagementSystem
